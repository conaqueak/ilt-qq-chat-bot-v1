## 项目简介

本项目是一个基于 Flask 框架开发的 QQ 机器人后端服务，目前主要用于群聊自动化信息推送和网站信息查询。

## 功能特性
- 群聊命令自动回复，支持 CQ 码格式图片和文本消息
- 实时调用第三方 API 获取游戏数据，自动转换为中文并格式化展示
- 包含路由注册、API 封装、工具函数等模块，易于扩展和维护
- 支持《最终幻想14》、《Apex 英雄》和《守望先锋》官方网站的新闻资讯查询
- 支持《Apex 英雄》玩家信息、地图轮换、猎杀者分数线及大师人数查询

## 快速开始
1. 安装依赖：
	```bash
	pip install -r requirements.txt
	```
2. 启动服务：
	```bash
	python app.py
	```
3. 按照群聊命令格式发送消息即可获取对应内容。

（留意：需要自行寻找并启动适合的QQ监听应用提供给此项目作为访问QQ内部数据的信息源接口，例如CQ-HTTP框架）
